Adding New Senior Player

  1. Log In
  2. Navigate to Player List Page

    3. You can do so by using:

    • The link in the dropdown menu in the header
    • The link in the dashboard
  3. Click on 'Add Player' button
  4. Fill in the required fields
  5. Submit

    6. You will be redirected back to the list after successful submit

    New player should appear on the bottom of the list

  6. That's it!
